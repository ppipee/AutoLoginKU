## Auto Login KU

[Login KU](https://login.ku.th)

### How to

- install package

```
yarn or npm i
```

#### Setup

- create file user.js
- create object that include "id" and "password"
- example

```javascript
module.exports = {
  id: 'username',
  password: 'password'
}
```

#### Run

```
node index.js
```
